# Network View and Discovery

## Network view

Consolidated view of relevant asks and offers. Filters by skill and status. Shows TTL countdown for urgency. Profile names displayed when available.

## Design

Text-first list view for beta instead of complex graph visualization. Future iterations may add full graph canvas built on the same data.

## Matching

Matching shows side-by-side ask/offer pairs. Filtering by skill and view type. Network views exclude expired entities by default.
