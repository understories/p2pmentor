# Overview

This section provides an overview of the p2pmentor beta documentation structure and how to navigate it.

## Documentation Structure

- **Introduction**: Project overview, getting started, and roadmap
- **Philosophy**: Core values and design principles
- **History**: Context and background
- **Architecture**: Technical architecture and design decisions
- **Arkiv**: Arkiv-specific implementation details and data model
- **Modules**: Reusable modules and packages
- **Practices**: Best practices and guidelines
- **User Flows**: User-facing features and workflows
- **Meta**: Documentation metadata and outline

## Getting Started

- **Builders**: Start with [Architecture](/docs/architecture/overview) and [Arkiv Integration](/docs/arkiv/overview)
- **Designers/PMs**: Start with [User Flows](/docs/user-flows/overview)
- **Users**: Start with [Getting Started](/docs/introduction/getting-started)

See [README](/docs/introduction/README) for the documentation entry point.
