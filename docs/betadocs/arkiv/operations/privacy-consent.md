# Privacy and Consent

## Explicit communication

Explicit communication that data is written to Arkiv testnet. Clear guidance that testnet wallets should not hold real funds.

## Feedback and telemetry

Feedback and telemetry features designed with opt in by default, not silent collection. All feedback stored on Arkiv, no opaque off-chain logging.

## Data ownership

Users own and control their data. The application is a client of a shared data layer, not the owner of data.
