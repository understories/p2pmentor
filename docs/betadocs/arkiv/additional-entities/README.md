# Additional Entities

Supporting entity types for observability, metrics, and system operations.

- [Admin Response](/docs/arkiv/additional-entities/admin-response) - Admin responses to user feedback
- [App Feedback](/docs/arkiv/additional-entities/app-feedback) - User feedback and issue reports
- [Auth Identity](/docs/arkiv/additional-entities/auth-identity) - Authentication and identity management
- [Beta Access](/docs/arkiv/additional-entities/beta-access) - Beta code access tracking
- [Beta Code](/docs/arkiv/additional-entities/beta-code) - Beta invite code management
- [Client Performance Metric](/docs/arkiv/additional-entities/client-perf-metric) - Client-side performance tracking
- [DX Metric](/docs/arkiv/additional-entities/dx-metric) - Developer experience metrics
- [Garden Note](/docs/arkiv/additional-entities/garden-note) - Public garden notes and comments
- [GitHub Issue Link](/docs/arkiv/additional-entities/github-issue-link) - Links between feedback and GitHub issues
- [Learning Follow](/docs/arkiv/additional-entities/learning-follow) - Skill following and learning interests
- [Metric Aggregate](/docs/arkiv/additional-entities/metric-aggregate) - Aggregated performance metrics
- [Notification](/docs/arkiv/additional-entities/notification) - User notifications
- [Notification Preferences](/docs/arkiv/additional-entities/notification-preferences) - User notification preferences
- [Onboarding Event](/docs/arkiv/additional-entities/onboarding-event) - Onboarding progress tracking
- [Performance Snapshot](/docs/arkiv/additional-entities/perf-snapshot) - Performance snapshots for comparison
- [Retention Metric](/docs/arkiv/additional-entities/retention-metric) - User retention tracking
- [Virtual Gathering](/docs/arkiv/additional-entities/virtual-gathering) - Virtual event management
